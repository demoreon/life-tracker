7. Daily vs Weekly Habits

EASY

By default, every habit shows every day. To make a habit show only on your review day (default: Sunday), add weekly: true.

Make a Habit Weekly

// Daily habit (shows every day):
{ name: "Exercise", cat: 3, time: 0, est: "20m" },

// Weekly habit (shows only on Sunday):
{ name: "Review subscriptions", cat: 0, time: 1, est: "10m", weekly: true },

Change a Weekly Habit Back to Daily

// Remove the weekly: true part:

// BEFORE (weekly):
{ name: "Review subs", cat: 0, time: 1, est: "10m", weekly: true },

// AFTER (daily):
{ name: "Review subs", cat: 0, time: 1, est: "10m" },

Change Which Day Weekly Habits Appear

// In the settings section at the top of data.js:

weeklyReviewDay: 0,  // 0=Sunday (default)
weeklyReviewDay: 6,  // 6=Saturday
weeklyReviewDay: 5,  // 5=Friday

9. Backup & Restore Data

MEDIUM

Your check-in data is saved in localStorage (your browser). It stays unless you clear browser data. Here's how to backup and restore:

Export (Backup)

Open your browser console (F12 â†’ Console tab) and paste:

const backup = JSON.stringify(localStorage);
console.log(backup);
// Copy the output and save it in a text file

Import (Restore)

Paste this in the console, replacing YOUR_BACKUP with your saved data:

const data = JSON.parse('YOUR_BACKUP_STRING_HERE');
Object.entries(data).forEach(([k, v]) => localStorage.setItem(k, v));
location.reload();

Clear All Data (Start Fresh)

localStorage.clear();
location.reload();
âš ï¸ Warning: localStorage.clear() deletes ALL your check-in history permanently.
